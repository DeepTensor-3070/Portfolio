NexReel: AI-Powered Auto Reel Generator

NexReel is an AI-powered Flask web application that automatically generates short social media reels from user-uploaded images and text descriptions. The system converts text into voice-over using Text-to-Speech and dynamically stitches images and audio together into a video using FFmpeg.

Designed as an end-to-end media automation pipeline, NexReel demonstrates backend development, media processing, and real-world content automation.

Loan Approval & Risk Assessment System

A machine learning–based loan management system that predicts loan approval decisions based on applicant financial and demographic data. The system helps financial institutions automate loan screening while minimizing default risk.

The project includes end-to-end data preprocessing, exploratory data analysis, feature engineering, and model evaluation using multiple classification algorithms. A web interface allows users to input applicant details and receive real-time predictions with confidence scores.
